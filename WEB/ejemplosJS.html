Función autofillInput: Esta función se ejecuta cuando el usuario pasa el ratón sobre el campo de entrada (inputText). Selecciona el campo de entrada usando document.getElementById('inputText'). Cambia el valor del campo de entrada a "Has passat per sobre el camp d'input!". Función clearInput: Esta función se ejecuta cuando el usuario hace clic en el botón "Clear". Selecciona el campo de entrada y registra su valor actual en la consola usando la función logEvent. Luego, limpia el campo de entrada estableciendo su valor a una cadena vacía (''). Función logEvent: Esta función registra mensajes en la consola. Recibe dos parámetros: elementType (el tipo de elemento) y message (el mensaje a registrar). Usa console.log para mostrar el mensaje en la consola. Event Listeners: document.getElementById('inputText').addEventListener('mouseover', autofillInput); Añade un evento mouseover al campo de entrada. Cuando el usuario pasa el ratón sobre él, se ejecuta la función autofillInput. document.getElementById('clearButton').addEventListener('click', clearInput); Añade un evento click al botón "Clear". Cuando el usuario hace clic en él, se ejecuta la función clearInput. Mejoras y Aclaraciones Uso de const y arrow functions: En el código, se usan const y funciones de flecha (=>), que son características modernas de JavaScript. Esto hace que el código sea más conciso y legible. Registro de Eventos: La función logEvent es útil para depurar y entender qué está sucediendo en el código. Registra el valor del campo de entrada antes de limpiarlo. Buenas Prácticas: Es una buena práctica separar el código JavaScript en funciones pequeñas y específicas, como autofillInput y clearInput. Esto hace que el código sea más modular y fácil de mantener.
